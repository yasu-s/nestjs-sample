## Description


## Installation

```bash
yarn
```

## Running the app

```bash
# development
yarn start

# watch mode
yarn start:dev

# production mode
yarn start:prod
```

### Debug URL

- Swagger - http://localhost:3000/api/
- Graphql - http://localhost:3000/graphql

## Test

```bash
# unit tests
yarn test

# test coverage
yarn test:cov
```

## Documents

```bash
yarn compodoc
```
